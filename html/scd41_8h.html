bool SCD41_init ( void )

Initializes the SCD41 sensor.

Stops any previous periodic measurement and starts a new periodic measurement cycle (approx 5s interval).

  • Returns
    true If the start command was acknowledged by the sensor.
    false If I2C communication failed.
    Initializes the SCD41 sensor.
