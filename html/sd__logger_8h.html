◆ SDLOG_init()

bool SDLOG_init ( void )

Initializes the SPI bus and the SD card.

This function initializes SPI with specific pins, attempts to mount the SD card, and prepares the log file. If the file is empty, it writes the CSV header.

  • Returns
    true If the SD card was initialized and the file is ready.
    false If the SD card mount failed or the file could not be opened.
    Initializes the SPI bus and the SD card.
